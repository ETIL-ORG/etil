# Windows 11 Workspace & Window Management Shortcuts

A quick reference guide for managing Virtual Desktops and Snap Layouts.

## 🖥️ Virtual Desktop (Workspace) Keys

| Shortcut | Action |
| :--- | :--- |
| **Win + Tab** | Open **Task View** (View all desktops and windows) |
| **Win + Ctrl + D** | Create a **new** virtual desktop |
| **Win + Ctrl + ← / →** | **Switch** between virtual desktops |
| **Win + Ctrl + F4** | **Close** the current virtual desktop |

## 📦 Moving Apps Between Desktops
*There is no single direct shortcut to move an app, but you can use these methods:*

1. **Task View Method**:
    - Press `Win + Tab`.
    - **Drag and drop** the window into a different desktop thumbnail at the bottom.
2. **Menu Method**:
    - Press `Win + Tab`.
    - **Right-click** the window and select **Move to** > **[Desktop Name]**.
3. **Keyboard-Only Method**:
    - Press `Win + Tab`.
    - Use **Arrow Keys** to highlight the app.
    - Press **Menu Key** (or `Shift + F10`) > Use arrows to select **Move to** > **Enter**.

## 📏 Snap Layouts & Window Tiling

| Shortcut | Action |
| :--- | :--- |
| **Win + Z** | Open the **Snap Layouts** menu for the active window |
| **Win + Arrow Keys** | Snap window to left, right, top, or bottom |
| **Win + Shift + ← / →** | Move active window to a **different monitor** |
| **Win + D** | Show or hide the desktop (Minimize/Restore all) |

## 💡 Quick Tips
* **Renaming**: In `Win + Tab`, right-click a desktop thumbnail to rename it (e.g., "Work," "Social").
* **Backgrounds**: You can set a unique wallpaper for each desktop to tell them apart easily.
